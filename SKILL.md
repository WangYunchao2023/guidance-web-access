# SKILL.md - Guidance Web Access (v3.2.0)

## 🌟 核心理念
- **Cortana 先理解 (Cortana First)**：接到任务后，Cortana 先进行任务分析，制定执行策略，再协调执行。
- **经验优先 (Experience First)**：系统会优先检索 `user_overrides.yaml`。若匹配任务关键词，将放弃默认路径，直奔"人工经验"锁定的高价值 URL。
- **智能化感知 (Interface Perception)**：不再硬编码输入框 ID。脚本会自动扫描页面，感知输入字段。
- **语义过滤 (Semantic Filter)**：智能过滤结果，确保只下载与任务高度相关的内容。

## 🤖 Cortana 任务理解流程（固化）

### 接到任务后，Cortana 先分析：

#### 1. 理解任务本质
- 用户要查找什么？（指导原则/法规）
- 核心实体是什么？（如"中药注射剂"拆分为"中药"+"注射剂"）
- 是否有日期限定？

#### 2. 制定搜索策略
- **主关键词**：完整匹配词
- **备选关键词**：截短词或同义词
- **过滤条件**：确保结果相关性的关键条件

#### 3. 检查人工经验匹配
- 检查 `user_overrides.yaml` 中是否有匹配的经验规则
- **有匹配** → 使用经验指定的 URL 和执行方式
- **无匹配** → 使用默认探索路径

#### 4. 生成完整执行方案
- 确定搜索词、过滤条件、执行方式
- 生成最优命令参数

### 执行示例

**用户**: "下载中药注射剂相关的指导原则"

**Cortana 分析**:
```
1. 任务理解
   - 核心实体: 中药 + 注射剂
   - 搜索词: "中药注射剂"
   - 过滤条件: ["中药", "注射剂"]

2. 经验匹配
   - 匹配规则: (.*)相关的指导原则
   - 经验方法: search_only
   - 经验URL: https://www.cde.org.cn/zdyz/fullsearchpage

3. 完整策略
   - 使用经验搜索页
   - 搜索词: "中药注射剂"
   - 结果过滤: 含"中药" AND 含"注射剂"
```

**执行**:
```bash
# 方式1：Cortana 主导模式（推荐）
python web_access.py --cortana-plan '{
  "task": "下载中药注射剂相关指导原则",
  "search_url": "https://www.cde.org.cn/zdyz/fullsearchpage",
  "search_var": "中药注射剂",
  "filter_criteria": ["中药", "注射剂"],
  "method": "search_only",
  "save_dir": "~/Documents/工作/法规指导原则/中药注射剂"
}'

# 方式2：传统模式（脚本自动分析）
python web_access.py "中药注射剂" --extra-filter "中药,注射剂" --save-dir "~/Documents/工作/法规指导原则/中药注射剂"
```

### 标准化报告格式

Cortana 分析后输出标准格式报告：

```
🎯 AI任务理解: 查找中药注射剂相关指导原则
📋 核心实体: 中药注射剂
🔍 搜索策略:
   1. "中药注射剂" (完整匹配)
   2. "中药" + 过滤"注射剂"
   3. "注射剂" + 过滤"中药"
📌 匹配经验: (.*)相关的指导原则 (search_only)
💡 建议: 按策略1→2→3顺序执行，保留有效结果
```

**说明**：
- `💡 建议` 为 Cortana 给出的最优执行建议
- 经验匹配部分应在策略制订后、实际执行前输出

## 🧠 AI 搜索与探索逻辑

### 1. 经验引导下的精准狙击
1. 检查 `references/user_overrides.yaml` 是否命中任务（如"指导原则"）。
2. **命中**：直接打开 `target_url` 或 `search_url`。
3. **未命中**：按 [首页] -> [发布通告] -> [业务专栏] 进行地毯式探测。

### 2. 界面感知与动作 (Sensing)
- **输入识别**：自动定位 `keyword`、`title`、`start_date`、`end_date` 等输入框。
- **多字段检索**：在搜索页同时填充"日期范围"与"关键词"，实现一键精准过滤。

### 3. 语义过滤 (Semantic Filter)
- **强匹配**：只有与任务日期（YYYY-MM-DD）完全一致的内容才会被保留下载。
- **噪音排除**：自动过滤党建、获奖、招聘等与研发无关的行政通告。

## 📂 存储与增量规范

### 文件保存
- **法规指导原则**：`~/Documents/工作/法规指导原则/`
- **监测更新任务**：`~/Documents/工作/法规指导原则更新/`
- **命名规范**：`发布日期 - 完整文件标题.pdf` (医药类专用)

### 增量去重
- 下载前检测当前保存目录下是否存在同名文件。若已存在，秒跳过。
